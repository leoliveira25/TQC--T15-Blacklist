# TQC--T15-Blacklist
Este repositório contém os casos de teste da squad blacklist.

Back-end (Postman): Permite que usuários administradores alterem o status de uma empresa (true = ativo, false = inativo), com registro de histórico e validação de permissões.

Front-end: Edição de centro de custo via tela com botões Editar, Salvar e Cancelar, garantindo persistência ou cancelamento das alterações.

Back-end (Interface): Login de usuário validando credenciais corretas e bloqueando acessos não autorizados.
